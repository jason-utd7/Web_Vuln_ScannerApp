Web Vulnerability Scanner

About-Us

This web application is created to help users detect common web vulnerabilities in websites. It scans for Cross-Site Scripting (XSS), Cross-Site Request Forgery (CSRF), Injection Flaws, SQL Injection, Open Redirect, and Server-Side Request Forgery (SSRF).

Severity Levels

In the circumstances of vulnerability scanning, the severity levels show the potential impact of a vulnerability. Here is what each severity level means:

  • Low: Low severity vulnerabilities generally have small impact and are relatively easy to fix.
  • Medium: Medium severity vulnerabilities have a hasty impact and may need more attention to fix.
  • High: High severity vulnerabilities have a significant impact and need attention asap to fix as they give a serious threat to the security of the system/website.
